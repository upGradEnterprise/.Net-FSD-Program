Steps to implement DB First Approach
~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~
Step 1: Create database & tables in SQL Server.
Step 2: Create ASP.NET Core MVC project.
Step 3: Install EF Core packages.
Step 4: Scaffold database with Scaffold-DbContext.

				Scaffold-DbContext 
							"Name=DefaultConnection" 
							Microsoft.EntityFrameworkCore.SqlServer 
							-OutputDir Models

Step 5: Register DbContext in Program.cs.
Step 6: Inject the Context class in Controller as dependent and  Use entities in controllers.
Step 7: Create views for CRUD.
Step 8: Run application & test.
Step 9: Re-scaffold when schema changes.

Syntax:

Entity(Table) Specific Scaffolding :
